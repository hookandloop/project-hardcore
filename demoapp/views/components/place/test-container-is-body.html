Place Test: Scrollable Body

Related JIRA Ticket. When an element is placed with the "Place" behavior, the placement algorithm needs to take scrolling parent containers correctly into account. In this test, the "body" element is designed to scroll. The Place behavior's calculation should factor in the body's "scrollHeight" when placing elements.

Scroll to the bottom of this panel to see timepickers in various places. When the timepickers' popups are opened, they should be positioned correctly against their input fields no matter where the container is scrolled.
