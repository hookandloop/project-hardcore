Toolbar Test: More Than 3 Buttons (With a popup)

Related JIRA Ticket. SoHo Xi Controls Guidelines state that there should be very few buttons present in the Toolbar Control, ideally no more than three including the spillover button. However, in some cases this guideline needs to be bent. This test makes sure the option that allows for this works properly.
