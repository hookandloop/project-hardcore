Mask Test: Copy/Paste

It should be possible for the Mask Control to handle input that was pasted from another source and properly format that input according to the Mask pattern's rules.

To manually simulate the test script:

  1. Select the contents of the "Basic Input Field" on the right
  2. Copy the text to the clipboard
  3. Paste the contents of the clipboard into the "Phone Number" field
The mask should properly format the contents of the field.
