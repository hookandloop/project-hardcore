Place Test: Nested Scrollable Container

Related JIRA Ticket. When an element is placed with the "Place" behavior, the placement algorithm needs to take scrolling parent containers correctly into account. In this test, the "body" and "page container" elements are fixed not to scroll, but two containers inside the "page container" are designed to scroll. These inner containers should be used in the Place behavior's calculation.

Scroll to the bottom of this panel to see a timepicker. When the timepicker's popup is opened, it should be positioned correctly against the input field no matter where the container is scrolled.
